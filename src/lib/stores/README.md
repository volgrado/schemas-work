# Stores

This directory contains all of the Svelte stores for the application. Svelte stores are used to manage the application's state in a way that is easy to share between components.

## Stores

- **`cardEditorStore.ts`**: Manages the state and interactions for the `CardEditorPanel` component.
- **`commandBarStore.ts`**: Manages the global state for the command bar and related UI modals.
- **`documentStore.ts`**: Manages the lifecycle, state, and persistence of the active schema document.
- **`editorStore.ts`**: Manages the global state of the Tiptap editor instance.
- **`reviewStore.ts`**: Manages the state and logic for the spaced repetition review feature.
- **`slashMenuStore.ts`**: Manages the state and interactions for the editor's slash command menu.
- **`ttsStore.ts`**: Manages the global state and orchestration for the Text-to-Speech (TTS) feature.
